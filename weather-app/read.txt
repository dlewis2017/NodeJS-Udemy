Must be on guest wifi
command line program using forecast.io
uses npm libraries
tells temperature uses given location
app.js uses callbacks with the request library
app-promise.js uses promises with the Axios library

usage:
npm init to install json package file with saved dependencies
node app.js -a "location"
node app-promise.js -address "location"
node app.js --help
node app-promise.js --h
